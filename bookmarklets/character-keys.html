This bookmarklet triggers a keypress event for every printable character (and then some) on a US standard 101 keyboard. The keypress details are logged to the console.

In practice, this bookmarket will trigger any single character key shortcut on a page or application, surfacing potential 2.1.4 Character Key Shortcuts (A) issues.

Related, Toptal has a helpful keycode reference table.

This test page contains a script that triggers an alert message when the p or P key is pressed. If the alerts appear when the bookmarklet is activated, the bookmarklet has successfully send those key presses.
